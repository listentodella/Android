# 抽象类
# 向下定义好框架，向上提供统一的接口
## 概念：含有纯虚函数的类
* virtual 函数声明时后面加上"= 0"
```
virtual void eating(void) = 0;
```
* 纯虚函数不需要定义
* 抽象类不能实例化对象
* 抽象类的子类如果没有复写全部的纯虚函数，则子类仍旧是抽象类
* 基类的析构函数不应该是纯虚函数
